Clock

JavaScript and Java 2D

JavaScript is a scripting, programming language, commonly used as part of web browsers. It allows client-side scripts to interact with the user, control the browser, communicate asynchronously, and alter the document content that is displayed. Implementations are also in server-side programming, game development, desktop and mobile applications.

This clock was similar to the one created for the class Midterm I took in Fall 2013, IS 545 - Advanced Internet Programming II. The principle is to to combine HTML, JavaScript and CSS to create an application. The functions used are “setInterval” function to call the "clock" function every second, the “clock” function to search for all elements and class names in the HTML that match the “figure” class and the element “date”, create an object called “Date” and sets it to the variable called “now”, acquire the time and change it to images and text.

A separate HTML page of a calendar previously designed for my JavaScript class was embedded into this page using iframe. An HTML5 iframe helps to reduce the risk of overlapping design of my overall website and puzzle's own page design, avoid formatting errors when both CSS (Cascading Style Sheets) files for each page combined, as each looks different. The iframe allows us to insert a HTML page into a website readily with less work, time, while reducing complexity and keeping the design of each page without much changes.

A real time analog clock made by Java 2D applet is shown below. The source code as a text file is available. This program uses the Timer class to set a period of 500 milliseconds and a panel object as the listener, which will trigger the actionPerformed method that gets system time and sets angles for three rotations of hour, minute, and second hands. After rotations are updated, the display is redrawn using the repaint method call.
