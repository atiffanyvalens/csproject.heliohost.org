Today's Sliding Blocks Puzzle

New Perspectives JavaScript and Ajax, Carey, Canovatchel, 2nd Edition, Tutorial 6, Review Assignments

In a sliding block puzzle, the puzzle pieces are laid out in a grid with one blank space. Pieces adjacent to the blank space (either above, below, to the left, or to the right of the space) can be swapped with the space. The goal of the puzzle is to move the blocks into their correct positions on the grid. Before beginning, please read the instructions on the right column of the puzzle. This game can be played using mouse or keyboard. Enjoy sliding blocks and happy playing.

The two JavaScript files used in making this puzzle are library.js and slideblocks.js. A separate HTML page of a sliding block previously designed for my JavaScript class was embedded into this page using iframe. An HTML5 iframe helps to reduce the risk of overlapping design of my overall website and puzzle's own page design, avoid formatting errors when both CSS (Cascading Style Sheets) files for each page combined, as each looks different. The iframe allows us to insert a HTML page into a website readily with less work, time, while reducing complexity and keeping the design of each page without much changes.
