Selection Sort

The purpose of sorting is to arrange data from lowest value to highest or vice versa. To understand selection sorting, assume that the items would be arranged from the lowest value on the left side to the highest value on the right. Take an example of a five element array with numbers from 1 to 5.

The data are in this order:                                                5, 3, 2, 4, 1.
The end result that we would look for should be:     1, 2, 3, 4, 5.

Selection Sort works as follows: first find the smallest in the array and exchange it with the element in the first position, then find the second smallest element and exchange it with the element in the second position, and continue in this way until the entire array is sorted. For example, using the array above, shown with array elements in sequence separated by commas:
5, 3, 2, 4, 1.

The leftmost element is at index zero, and the rightmost element is at the highest array index, in our case, 4 (the effective size of our array is 5). The smallest element in this effective array (index 0-4) is at index 4. We have shown the smallest element and the one at the lowest index in bold. We then swap the element at index 0 with that at index 4. The result is:
1,3,2, 4, 5.


The first element in the array, 1, is already sorted. The effective size of the above array is reduced to 4, and the first element on the most left side will not be included in the next iteration, making the highest index in the effective array now 3. The smallest element in this effective array (index 0-3) is 2 at index 1, so we swap elements at index 1 and 0 (in bold) and the array becomes:
1, 2, 3, 4, 5.

Before continuing to next iteration, we realize that the list of numbers is sorted and the Selection Sort algorithm is complete.
Properties of selection sort are:   Not stable, O(1) extra space, Θ(n2) comparisons, Θ(n) swaps, Not adaptive.

Selection Sort Algorithm

public class selectionsort {

static void selectionSort ( Object[] data) {
         int i,j,least;
         for (i = 0; i < data.length-1; i++) {
                  for (j = i+1, least = i; j < data.length; j++) {
                       if (((Comparable) data[j]).compareTo (data[least]) < 0)
                             least = j;
                       if (i != least)
                             swap(data, least, i);
                  }
          }
}

static void swap ( Object[] a, int e1, int e2) {
         Object tmp = a[e1];
         a[e1] = a[e2];
         a[e2] = tmp;
}
}
