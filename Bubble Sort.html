Bubble Sort

The purpose of sorting is to arrange data from lowest value to highest or vice versa. To understand bubble sorting, assume that the items would be arranged from the lowest value on the left side to the highest value on the right. Take an example of a five element list with numbers from 1 to 5.

The data are in this order:                                                5, 3, 2, 4, 1.
The end result that we would look for should be:     1, 2, 3, 4, 5.
Bubble sorting begins by comparing each pair of adjacent items. If one of those two items is of smaller value (lighter) and if the lighter element is on the right side of its neighbour, they swap places. When this element encounters another element which is lighter and it is on the left side, the lighter item would then be compared with another item on its left side. Element moves to the left side when it is smaller in a way that Bubble sorting makes a lighter item appears to bubble up to the surface, and the lightest would be swapped continuously until it reaches the most left side. The process would be repeated until all elements in the list are sorted.

Bubble Sort Algorithm

public class bubblesort {

private static void bubbleSort(int[] intArray) {
          int n = intArray.length;
          int temp = 0;
          for  ( int i = 0;     i < n ;     i++ ) {
                 for  ( int j = 1;     j <(n-i);    j++ ){
                       if (intArray[j-1] > intArray[j] ){
                             // swap the elements!
                             temp = intArray[j-1];
                             intArray[j-1] = intArray[j];
                             intArray[j] = temp;
                       }
                 }
           }
}
}
