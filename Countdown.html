Countdown to Upcoming Events

New Perspectives JavaScript and Ajax, Carey, Canovatchel, 2nd Edition, Tutorial 2, Review Assignments

JavaScript is a scripting, programming language, commonly used as part of web browsers. It allows client-side scripts to interact with the user, control the browser, communicate asynchronously, and alter the document content that is displayed. Implementations are also in server-side programming, game development, desktop and mobile applications. This page contains a list of special events that includes each event's name, the date that it will occur, and the time remaining until the event. The contents are constantly updated by a user's browser which include a countdown clock with current date and time.

The JavaScript file used in making this calendar is dates.js. A separate HTML page of this countdown previously designed for my JavaScript class was inserted into this page using iframe. An HTML5 iframe helps to reduce the risk of overlapping design of my overall website and puzzle's own page design, avoid formatting errors when both CSS (Cascading Style Sheets) files for each page combined, as each looks different. The iframe allows us to insert a HTML page into a website readily with less work, time, while reducing complexity and keeping the design of each page without much changes.
