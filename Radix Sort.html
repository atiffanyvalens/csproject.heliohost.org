Radix Sort

Radix sort is a small method that many people intuitively use when alphabetizing a large list of names. (Here Radix is 26, 26 letters of alphabet). Specifically, the list of names is first sorted according to the first letter of each names, that is, the names are arranged in 26 classes. In sorting other data type such as numbers, one might want to begin with their most significant digit. But Radix sort do counter-intuitively by sorting on the least significant digits first. On the first pass entire numbers will be arrange on the least significant digit and combined in a array. Then on the second pass, the entire numbers are sorted again on the second least-significant digits and combine in a array and so on.

Following example shows how Radix sort operates on seven 3-digits number.

It is a great tutorial which highlights that Radix Sort is a non-comparative sorting method that seems to magically sort an array of numbers without comparing the value of a member to one another, but by grouping them into bins ranging from 0 to 9, sorting from their digits of tenthousandth, thousandth, hundredth, and so on, and then rearranging the numbers continuously until the array is sorted.



Radix Sort Algorithm

public void radixSort(int[] a){

int i, m = a[0], exp = 1, n = a.length;
int[] b = new int[10];

for (i = 1; i < n; i++)
if (a[i] > m)
        m = a[i];

while (m / exp > 0) {

int[] bucket = new int[10];

for (i = 0; i < n; i++)
        bucket[(a[i] / exp) % 10]++;

for (i = 1; i < 10; i++)
        bucket[i] += bucket[i - 1];

for (i = n - 1; i >= 0; i--)
        b[--bucket[(a[i] / exp) % 10]] = a[i];

for (i = 0; i < n; i++)
        a[i] = b[i];

exp = exp* 10;
}
}
